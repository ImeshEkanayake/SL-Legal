# SL-Legal

Public repository for the Sri Lankan Legal Assist project split.

## Folders

- `V1-Evidence-document-extraction`: frozen V1 baseline for evidence document extraction.
- `V2-For-Against-Docs`: V2 development copy for claim-level for/against document analysis.

## Data

The large legal corpus data and generated tracking CSVs are not included in this repository yet. Data upload will be planned separately using an appropriate storage strategy.

## Database

Both versions are intended to use the existing database configuration. No database schema changes are included in this upload.
